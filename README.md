## 规则解析？

现在当 validate 整个 form 的时候，会遍历每一个名字，然后调用各自的 validateInput，那么，抽取出的验证模块（纯验证，与UI无关只能验证一个 name）

## 问题

对于异步验证，举个例子，加入是 `onChange` 触发验证，那么对于频繁修改表单，会不断地触发回调，造成显示上出现验证和验证结束UI之间的闪烁
（初步解决方案，通过观察现在的值跟之前放进去的值是否一致来决定是否触发回调）
